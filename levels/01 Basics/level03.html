Bis jetzt haben wir sehr einfache Pakete gesehen. Diese Pakete zeigten nur die Quelle (wer das Paket gesendet hat) und das Ziel (wer das Paket empfangen hat). In diesem Level sehen wir ein komplexeres Paket, das dem Empfänger Informationen darüber gibt, wie er antworten soll.

Pausiere die Simulation, um das Paket anzusehen, das von Alices Computer an Wikipedia gesendet wird. Es gibt einen neuen Header in der Transportschicht. Der Header proto definiert das Protokoll, das auf dieser Schicht ausgeführt wird. Das aufgeführte Protokoll ist ICMP, welches verwendet wird, um Computer anzupingen. Wenn ein Computer ein Paket mit einer ICMP-Echo-Anfrage (Ping) erhält, antwortet er, indem er ein Paket mit einem ICMP-Echo (Pong) zurücksendet. Setze die Simulation fort und achte auf das Antwortpaket, das von Wikipedia an Alices Computer gesendet wird.

Levelziele

  • Um dieses Level abzuschließen, musst du 4 weitere ICMP-Echo-Anfragen von Alice an Wikipedia senden. Gehe vor wie zuvor, indem du auf das + klickst, um neue Pakete hinzuzufügen. Denk daran, das neue Transportschicht-Headerfeld „proto“ auszufüllen.
