Ein Server oder Computer kann sich schützen, indem er bösartige Computer mithilfe einer Firewall blockiert. Eine Firewall führt eine Liste „schlechter“ Quelladressen und verwirft alle Pakete, die von einer dieser Adressen stammen. In diesem Beispiel hat Wikipedia aus dem vorherigen Angriff gelernt, dass Alice eine bösartige Nutzerin ist, und hat eine Firewall eingerichtet, um sie zu blockieren. Beobachte, wie die Pakete von Alices Computer zu Wikipedia verworfen werden.

Eine Möglichkeit für einen Angreifer, eine Firewall zu umgehen, besteht darin, einen Distributed Denial of Service (DDoS)-Angriff zu starten. Dazu sendet der Angreifer Pakete von mehreren Maschinen, die unter seiner Kontrolle stehen.

Um dieses Level zu bestehen, starte einen DDoS-Angriff gegen Wikipedia von Alices zusätzlichen Maschinen.

Levelziele

  • Sende Wikipedia mehr Pakete, als es verarbeiten kann
