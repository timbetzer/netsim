In diesem Level wird Alices Internetnutzung zensiert. Sie kann eine erlaubte Seite besuchen, aber der Zensor verwirft alle Pakete zu einer blockierten Seite.

Eine Möglichkeit, Zensur zu umgehen, besteht darin, eine Zwischenverbindung zu einem Proxy-Server herzustellen. Dieser Server nimmt Pakete entgegen, die an ihn adressiert sind, und leitet sie zur blockierten Seite weiter. Da der Zensor nur die Verbindung zwischen Alice und dem Proxy sieht, weiß er nicht, welche Seite Alice tatsächlich besucht.

Um dieses Level zu bestehen, musst du den Zensor umgehen und ein Paket von Alice zur blockierten Seite senden.

Levelziele

  • Sende ein Paket von Alice zur blockierten Seite.
